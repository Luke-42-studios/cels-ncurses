  <action>
  Add `#include <signal.h>` and `#include <panel.h>` to the includes at the top (after the existing `#include <cels_ncurses_draw.h>`). Also add `#include "tui_window.h"` for the `tui_hook_frame_end()` declaration.

  1. **Add clearing to TUI_LayerSystem**: In the existing `cel_each` loop, after the `ncurses_layer_sync_visibility()` call and before the panel collection block, add clearing for visible layers:
     ```c
     /* Auto-clear visible layers (developer gets blank canvas at OnRender) */
     if (TUI_LayerConfig->visible) {
         ncurses_layer_clear_window(
             TUI_DrawContext_Component->win,
             TUI_DrawContext_Component->subcell_buf);
     }
     ```

  2. **Define TUI_FrameBeginSystem** (after TUI_LayerSystem, before the Module Init section). This system blocks SIGWINCH during rendering. Must be defined AFTER TUI_LayerSystem so it registers after it (CELS runs same-phase systems in registration order):
     ```c
     CEL_System(TUI_FrameBeginSystem, .phase = PreRender) {
         cel_run {
             sigset_t winch_set;
             sigemptyset(&winch_set);
             sigaddset(&winch_set, SIGWINCH);
             sigprocmask(SIG_BLOCK, &winch_set, NULL);
         }
     }
     ```

  3. **Define TUI_FrameEndSystem** (after TUI_FrameBeginSystem, before Module Init). This system composites panels, flushes to terminal, unblocks SIGWINCH, and throttles FPS:
     ```c
     CEL_System(TUI_FrameEndSystem, .phase = PostRender) {
         cel_run {
             update_panels();
             doupdate();

             sigset_t winch_set;
             sigemptyset(&winch_set);
             sigaddset(&winch_set, SIGWINCH);
             sigprocmask(SIG_UNBLOCK, &winch_set, NULL);

             tui_hook_frame_end();
         }
     }
     ```

  4. **Update CEL_Module(NCurses, init)**: Remove the `ncurses_register_frame_systems()` call. Add `TUI_FrameBeginSystem` and `TUI_FrameEndSystem` to the `cels_register()` call (append after `TUI_LayerSystem`):
     ```c
     CEL_Module(NCurses, init) {
         cels_register(NCurses_WindowState, NCurses_InputState,
                       NCursesWindowLC,
                       TUI_Renderable, TUI_LayerConfig, TUI_DrawContext_Component,
                       TUI_LayerLC, TUI_LayerSystem,
                       TUI_FrameBeginSystem, TUI_FrameEndSystem);
     }
     ```

  IMPORTANT ordering: TUI_LayerSystem -> TUI_FrameBeginSystem in both definition order and cels_register order. This ensures layers are cleared/synced BEFORE SIGWINCH is blocked, which is the correct sequence (Pitfall 2 from research).

  Do NOT add any timing/delta_time code to frame systems -- WindowState already handles that (Pitfall: duplicate timing).
  Do NOT add g_layers[] clearing -- that's the legacy system being removed in Plan 02.
  </action>
