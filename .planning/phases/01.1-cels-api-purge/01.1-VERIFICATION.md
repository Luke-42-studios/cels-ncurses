# Phase 1.1: CELS API Purge Verification Report

**Phase Goal:** Eliminate all direct flecs/ecs API usage from cels-ncurses source files. All observer registration, system registration, component manipulation, and world access must go through CELS abstractions (CEL_System, cel_query, cels_ensure_component, cels_component_add, etc.). No `#include <flecs.h>` in any cels-ncurses source file outside of `#ifdef` guards for features CELS doesn't yet wrap.
