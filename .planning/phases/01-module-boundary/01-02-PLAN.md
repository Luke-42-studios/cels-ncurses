Rewrite tui_window.c to replace the backend-descriptor hook pattern with flecs observers. Preserve ALL ncurses implementation logic -- only the wrapping changes.

**What to keep (copy from existing tui_window.c):**
- Signal handler: tui_sigint_handler
- Cleanup: cleanup_endwin with atexit
- ALL of tui_hook_startup logic (locale, initscr, cbreak, noecho, keypad, curs_set, nodelay, ESCDELAY, color init via tui_color_init)
- ALL of tui_hook_shutdown logic (endwin)
- Frame timing logic from tui_hook_frame_begin (delta_time calculation via clock_gettime)
- FPS throttle from tui_hook_frame_end (usleep-based)
- Delta time getter

**What to remove:**
- Engine_WindowState global and Engine_WindowState_register
- TUI_Window_use function (provider registration pattern)
- CELS_BackendDesc struct and tui_backend_desc variable
- All references to CEL_Window, CEL_WindowID, CELS_WindowState, cels_backend_register, cels_window_set
- All references to `<cels/backend.h>` include
- The g_cel_window_entity and CEL_Window singleton code
- tui_window_get_standard_state function

**What to add:**

Includes:
- `<cels-ncurses/tui_ncurses.h>` (for NCurses_WindowConfig/State types and their _id variables)
- `<cels-ncurses/tui_internal.h>` (so the function signatures match)
- `<flecs.h>`, `<ncurses.h>`, `<cels/cels.h>`, standard lib headers

Static state:
- `static cels_entity_t g_window_entity = 0;` -- tracks the single window entity
- `static volatile int g_running = 1;` -- quit flag (keep)
- `static int g_ncurses_active = 0;` -- terminal active flag (keep)
- `static int g_target_fps = 60;` -- store fps from config
- Frame timing statics (keep from existing: g_frame_start, g_prev_frame_start, g_delta_time)

**ncurses_terminal_init(NCurses_WindowConfig* config)** -- static function:
- Extract the guts of tui_hook_startup:
  - signal(SIGINT, handler), atexit(cleanup_endwin)
  - setlocale, initscr, cbreak, noecho, keypad, curs_set, nodelay, ESCDELAY
  - Color init if has_colors()
  - Store target fps from config->fps (default 60)
  - Set g_ncurses_active = 1

**ncurses_terminal_shutdown()** -- static function:
- Extract the guts of tui_hook_shutdown:
  - If g_ncurses_active && !isendwin(): endwin(), g_ncurses_active = 0
  - g_window_entity = 0

**on_window_config_set(ecs_iter_t* it)** -- static observer callback:
- Following 01-RESEARCH.md Pattern 2 exactly:
  - Check if g_window_entity != 0: if so, fprintf(stderr, "[NCurses] Warning: only one window entity allowed, ignoring\n") and return
  - Get NCurses_WindowConfig* via ecs_field(it, NCurses_WindowConfig, 0)
  - For i = 0 to it->count: store g_window_entity = it->entities[i]
  - Call ncurses_terminal_init(&configs[i])
  - Call tui_frame_init() -- frame pipeline needs terminal active
  - Attach NCurses_WindowState to the entity via ecs_set_id: width = COLS, height = LINES, running = true, actual_fps = (float)target_fps, delta_time = 1.0f/fps
  - Call cels_request_quit() equivalent setup: nothing needed yet, g_running handles it

**on_window_config_remove(ecs_iter_t* it)** -- static observer callback:
  - For i = 0 to it->count: if it->entities[i] == g_window_entity, call ncurses_terminal_shutdown()

**ncurses_register_window_observer(void)** -- exported function:
- Get world from cels_get_world(cels_get_context())
- Create observer with ecs_observer_desc_t following RESEARCH.md Pattern 2:
  - Name: "NCurses_WindowConfigObserver"
  - Term: NCurses_WindowConfig_id
  - Event: EcsOnSet
  - Callback: on_window_config_set

**ncurses_register_window_remove_observer(void)** -- exported function:
- Same pattern but event = EcsOnRemove, callback = on_window_config_remove
- Name: "NCurses_WindowConfigRemoveObserver"

**tui_window_get_running_ptr(void)** -- keep exported (input system needs it for quit signaling):
- Return &g_running

**ncurses_window_update_state(void)** -- exported function for frame_begin to call:
- Replaces tui_hook_frame_begin dimension update logic
- Checks COLS/LINES against cached values, updates NCurses_WindowState on entity if changed
- Updates delta_time and actual_fps on the entity each frame
- This function is called from tui_frame.c's frame_begin system

**Update tui_internal.h** -- add these declarations after the existing 4 registration function declarations:
- `extern void ncurses_window_update_state(void);`
- `extern volatile int* tui_window_get_running_ptr(void);`

IMPORTANT: Do NOT include the old tui_window.h. Include tui_ncurses.h instead. The old tui_window.h will be deleted in Plan 03.

**src/input/tui_input.c refactor:**

2. Keep ALL existing code: g_input_state, tui_read_input_ncurses, tui_input_system_callback, mouse handling, quit guard, custom key codes, tui_pause, getters (tui_input_get_state, tui_input_get_mouse_position)
3. RENAME TUI_Input_use to ncurses_register_input_system:
   - Remove the TUI_Input config parameter (no longer provider-based)
   - Keep the function body: get running pointer, define_key for ctrl/shift arrows, init mouse state, mousemask, register ECS system at OnLoad
   - The running pointer: call tui_window_get_running_ptr() to get g_running from tui_window.c (keep this dependency -- input needs to signal quit)
   - Rename the ECS system entity from "TUI_InputSystem" to "NCurses_InputSystem"
4. Remove the old TUI_Input_use function entirely
5. The resize handler (KEY_RESIZE case): change `TUI_WindowState.width` reference to use COLS/LINES directly (the old reference to TUI_WindowState was just for the debug fprintf -- change the fprintf to use COLS/LINES or remove the old dimension reference)

**src/frame/tui_frame.c refactor:**

1. Keep ALL existing code: g_frame_state, g_background_layer, tui_frame_init, tui_frame_begin, tui_frame_end, tui_frame_invalidate_all, tui_frame_get_background, tui_frame_ensure_background, timing helpers
2. Keep the ECS system callbacks (tui_frame_begin_callback, tui_frame_end_callback)
3. RENAME tui_frame_register_systems to ncurses_register_frame_systems:
   - Same body, just rename for consistency
   - Rename system entities from "TUI_FrameBeginSystem"/"TUI_FrameEndSystem" to "NCurses_FrameBeginSystem"/"NCurses_FrameEndSystem"
4. Remove the `#ifdef CELS_HAS_ECS` guard around registration -- ECS is always present now (it's a CELS module)
5. Add a call to ncurses_window_update_state() at the start of tui_frame_begin_callback (before tui_frame_begin), so window state gets updated each frame
6. Add include: `<cels-ncurses/tui_internal.h>` (for ncurses_window_update_state declaration)

**include/cels-ncurses/tui_input.h update:**

1. Remove `#include <cels/cels.h>` -- replace with `#include <cels-ncurses/tui_ncurses.h>` (which includes cels.h)
2. Keep TUI_MouseButton enum and TUI_InputState struct as the implementation type. The g_input_state global uses TUI_InputState. The NCurses_InputState ECS component (defined in tui_ncurses.h) is a separate concern for Phase 3 -- do NOT add a typedef or bridge between them in this phase.
3. Remove TUI_Input_use declaration and TUI_Input config struct
4. Remove backward compatibility section (CELS_Input typedef, cels_input_get inline)
5. Keep: g_input_state extern, tui_input_get_state, tui_input_get_mouse_position, tui_input_set_quit_guard

2. Keep all public API: tui_frame_init, tui_frame_begin, tui_frame_end, tui_frame_invalidate_all, tui_frame_get_background, TUI_FrameState, g_frame_state extern
