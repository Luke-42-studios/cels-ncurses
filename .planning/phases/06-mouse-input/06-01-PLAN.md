<task type="auto">
  <name>Task 1: Define TUI_InputState struct and mouse types in tui_input.h</name>
  <files>include/cels-ncurses/tui_input.h</files>
  <action>
Rewrite `tui_input.h` to define the module-local input state, replacing all dependency on the removed `CELS_Input` type. Follow the exact extern-global pattern used by `TUI_FrameState g_frame_state` in `tui_frame.h`.

**Add these types:**

1. `TUI_MouseButton` enum: `TUI_MOUSE_NONE = 0`, `TUI_MOUSE_LEFT = 1`, `TUI_MOUSE_MIDDLE = 2`, `TUI_MOUSE_RIGHT = 3`

2. `TUI_InputState` struct containing ALL fields from the old CELS_Input PLUS new mouse fields:

   Keyboard fields (preserve v1.0 behavior):
   - `float axis_left[2]` -- directional input (-1.0/0.0/1.0 for x/y)
   - `bool button_accept` -- Enter/Return
   - `bool button_cancel` -- Escape
   - `bool key_tab`, `key_shift_tab`
   - `bool key_home`, `key_end`, `key_page_up`, `key_page_down`
   - `bool key_backspace`, `key_delete`
   - `int key_number` + `bool has_number`
   - `int key_function` + `bool has_function`
   - `int raw_key` + `bool has_raw_key`

   Mouse fields (new):
   - `int mouse_x`, `mouse_y` -- current mouse position in screen-relative cell coordinates (-1 = no position yet)
   - `TUI_MouseButton mouse_button` -- which button had an event this frame (NONE if no button event)
   - `bool mouse_pressed` -- true if mouse_button was pressed this frame
   - `bool mouse_released` -- true if mouse_button was released this frame
   - `bool mouse_left_held`, `mouse_middle_held`, `mouse_right_held` -- persistent held state (set on press, cleared on release)

3. Extern declaration: `extern TUI_InputState g_input_state;`

4. Getter functions:
   - `extern const TUI_InputState* tui_input_get_state(void);` -- returns pointer to g_input_state (read-only)
   - `extern void tui_input_get_mouse_position(int *x, int *y);` -- convenience getter for mouse position

**Keep existing declarations:**
   - `extern void TUI_Input_use(TUI_Input config);` (provider registration, unchanged)
   - `extern void tui_input_set_quit_guard(bool (*guard_fn)(void));` (quit guard, unchanged)
   - `typedef struct TUI_Input { int _placeholder; } TUI_Input;` (provider config, unchanged)

**Remove:** The `#include <cels/cels.h>` is still needed for `CEL_Use` macro support. Keep it. But there is NO include of `<cels/backend.h>` in this header (it was never here, but make sure it stays out).

**Add backward compatibility typedef:**
   ```c
   /* Backward compatibility -- consumers using CELS_Input can migrate */
   typedef TUI_InputState CELS_Input;
   ```

This typedef allows cels-widgets and space_invaders code to keep compiling while they migrate to the new name. Also add:
   ```c
   /* Backward compatibility -- consumers using cels_input_get() */
   static inline const CELS_Input* cels_input_get(void* ctx) {
       (void)ctx;
       return &g_input_state;
   }
   ```

This provides the `cels_input_get()` function that consumers call, redirecting to the module-local global. The `ctx` parameter is accepted but ignored (matching the old API signature).

<task type="auto">
  <name>Task 2: Implement mouse-enabled input system in tui_input.c</name>
  <files>src/input/tui_input.c</files>
  <action>
Rewrite `tui_input.c` to use the new `TUI_InputState g_input_state` instead of `CELS_Input` + `cels_input_set()`.

**Changes to static state section:**
- Define `TUI_InputState g_input_state = {0};` (the extern global)
- Add `static int g_mouse_initialized = 0;` flag
- Keep `g_tui_running_ptr` and `g_quit_guard_fn` unchanged

**Changes to TUI_Input_use() (initialization):**

After the existing `define_key()` calls, add mouse initialization:
```c
/* Enable mouse events: buttons 1-3 press/release + position tracking */
mmask_t desired = BUTTON1_PRESSED | BUTTON1_RELEASED
                | BUTTON2_PRESSED | BUTTON2_RELEASED
                | BUTTON3_PRESSED | BUTTON3_RELEASED
                | REPORT_MOUSE_POSITION;
mousemask(desired, NULL);
mouseinterval(0);  /* Disable click detection -- raw press/release only */
g_mouse_initialized = 1;
```

Do NOT add `printf("\033[?1003h")` escape sequences. Start with `mousemask()` only. The research notes this should be sufficient and escape sequences are a fallback if testing reveals issues.

**Changes to tui_read_input_ncurses():**

1. Replace `CELS_Input input; memset(&input, 0, sizeof(input));` with:
   ```c
   /* Reset per-frame fields (preserve persistent state like mouse_x/y and held buttons) */
   g_input_state.axis_left[0] = 0.0f;
   g_input_state.axis_left[1] = 0.0f;
   g_input_state.button_accept = false;
   g_input_state.button_cancel = false;
   g_input_state.key_tab = false;
   g_input_state.key_shift_tab = false;
   g_input_state.key_home = false;
   g_input_state.key_end = false;
   g_input_state.key_page_up = false;
   g_input_state.key_page_down = false;
   g_input_state.key_backspace = false;
   g_input_state.key_delete = false;
   g_input_state.has_number = false;
   g_input_state.has_function = false;
   g_input_state.has_raw_key = false;
   g_input_state.mouse_button = TUI_MOUSE_NONE;
   g_input_state.mouse_pressed = false;
   g_input_state.mouse_released = false;
   /* NOTE: mouse_x, mouse_y, mouse_*_held are NOT reset -- they persist across frames */
   ```

2. Remove all `CELS_Context* ctx = cels_get_context();` and `cels_input_set(ctx, &input);` calls. The function now writes directly to `g_input_state`. All the `return;` early-exit paths that previously called `cels_input_set` before returning should just `return;` (the state is already in g_input_state).

3. Replace all `input.` references with `g_input_state.` throughout the switch statement. For example:
   - `input.axis_left[1] = -1.0f;` becomes `g_input_state.axis_left[1] = -1.0f;`
   - `input.button_accept = true;` becomes `g_input_state.button_accept = true;`
   - etc.

4. Add `case KEY_MOUSE:` handler in the switch, placed BEFORE the `case KEY_RESIZE:` block:
   ```c
   case KEY_MOUSE: {
       /* Drain ALL queued mouse events -- keep final position, last button event.
        * This prevents position lag when mouse moves fast (many events per frame). */
       MEVENT event;
       do {
           if (getmouse(&event) != OK) break;

           /* Always update position (screen-relative cell coordinates) */
           g_input_state.mouse_x = event.x;
           g_input_state.mouse_y = event.y;

           /* Decode button press/release events */
           if (event.bstate & BUTTON1_PRESSED) {
               g_input_state.mouse_button = TUI_MOUSE_LEFT;
               g_input_state.mouse_pressed = true;
               g_input_state.mouse_released = false;
               g_input_state.mouse_left_held = true;
           } else if (event.bstate & BUTTON1_RELEASED) {
               g_input_state.mouse_button = TUI_MOUSE_LEFT;
               g_input_state.mouse_pressed = false;
               g_input_state.mouse_released = true;
               g_input_state.mouse_left_held = false;
           } else if (event.bstate & BUTTON2_PRESSED) {
               g_input_state.mouse_button = TUI_MOUSE_MIDDLE;
               g_input_state.mouse_pressed = true;
               g_input_state.mouse_released = false;
               g_input_state.mouse_middle_held = true;
           } else if (event.bstate & BUTTON2_RELEASED) {
               g_input_state.mouse_button = TUI_MOUSE_MIDDLE;
               g_input_state.mouse_pressed = false;
               g_input_state.mouse_released = true;
               g_input_state.mouse_middle_held = false;
           } else if (event.bstate & BUTTON3_PRESSED) {
               g_input_state.mouse_button = TUI_MOUSE_RIGHT;
               g_input_state.mouse_pressed = true;
               g_input_state.mouse_released = false;
               g_input_state.mouse_right_held = true;
           } else if (event.bstate & BUTTON3_RELEASED) {
               g_input_state.mouse_button = TUI_MOUSE_RIGHT;
               g_input_state.mouse_pressed = false;
               g_input_state.mouse_released = true;
               g_input_state.mouse_right_held = false;
           }
           /* else: motion-only event (position already updated above) */

       } while ((ch = wgetch(stdscr)) == KEY_MOUSE);

       /* Put back any non-mouse key for the next frame */
       if (ch != ERR) ungetch(ch);
       return;  /* Mouse events fully processed */
   }
   ```

5. Remove the `#include <cels/backend.h>` include. Keep all other includes. The function no longer uses any cels/backend.h types.

**Implement getter functions:**
```c
const TUI_InputState* tui_input_get_state(void) {
    return &g_input_state;
}

void tui_input_get_mouse_position(int *x, int *y) {
    if (x) *x = g_input_state.mouse_x;
    if (y) *y = g_input_state.mouse_y;
}
```
