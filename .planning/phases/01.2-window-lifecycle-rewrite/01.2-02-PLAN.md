3. **CEL_Lifecycle** -- window lifecycle:

4. **CEL_Observe on_create** -- terminal initialization:
   ```c
   CEL_Observe(NCursesWindowLC, on_create) {
       if (ncurses_window_get_entity() != 0) {
           fprintf(stderr, "[NCurses] Warning: only one window entity allowed\n");
           return;
       }
       ncurses_window_set_entity(entity);

       /* Read config from entity using CELS public API */
       cels_ensure_component(&NCurses_WindowConfig_id, "NCurses_WindowConfig",
                             sizeof(NCurses_WindowConfig), CELS_ALIGNOF(NCurses_WindowConfig));
       cels_ensure_component(&NCurses_WindowState_id, "NCurses_WindowState",
                             sizeof(NCurses_WindowState), CELS_ALIGNOF(NCurses_WindowState));

       const NCurses_WindowConfig* config =
           (const NCurses_WindowConfig*)cels_entity_get_component(entity, NCurses_WindowConfig_id);
       if (!config) {
           fprintf(stderr, "[NCurses] Error: on_create fired but no WindowConfig on entity\n");
           ncurses_window_set_entity(0);
           return;
       }

       ncurses_terminal_init((NCurses_WindowConfig*)config);

       /* Set initial WindowState on entity */
       NCurses_WindowState state = {
           .width = COLS,
           .height = LINES,
           .running = true,
           .actual_fps = (float)config->fps,
           .delta_time = 1.0f / (config->fps > 0 ? (float)config->fps : 60.0f)
       };
       cels_entity_set_component(entity, NCurses_WindowState_id,
                                  &state, sizeof(NCurses_WindowState));
       cels_component_notify_change(NCurses_WindowState_id);
   }
   ```
   Include `<stdio.h>` for fprintf.

5. **CEL_Observe on_destroy** -- terminal shutdown:
   ```c
   CEL_Observe(NCursesWindowLC, on_destroy) {
       (void)entity;
       ncurses_terminal_shutdown();
       ncurses_window_set_entity(0);
   }
   ```

6. **CEL_Module(NCurses)** -- simplified init body:
   ```c
   CEL_Module(NCurses) {
       /* Register component types */
       cels_ensure_component(&NCurses_WindowConfig_id, "NCurses_WindowConfig",
                             sizeof(NCurses_WindowConfig), CELS_ALIGNOF(NCurses_WindowConfig));
       cels_ensure_component(&NCurses_WindowState_id, "NCurses_WindowState",
                             sizeof(NCurses_WindowState), CELS_ALIGNOF(NCurses_WindowState));

       /* Register lifecycle (observers auto-register via constructor attribute) */
       NCursesWindowLC_register();

       /* Register systems (frame pipeline) */
       ncurses_register_input_system();
       ncurses_register_frame_systems();
   }
   ```
   Remove calls to `ncurses_register_window_observer()` and `ncurses_register_window_remove_observer()` -- replaced by CEL_Observe auto-registration.

7. **CEL_Compose(NCursesWindow)** -- add lifecycle binding:
   ```c
   CEL_Compose(NCursesWindow) {
       cel_has(NCurses_WindowConfig,
           .title = cel.title,
           .fps = cel.fps,
           .color_mode = cel.color_mode
       );
       /* Bind entity to window lifecycle -- fires on_create observer */
       cels_lifecycle_bind_entity(NCursesWindowLC_id, cels_get_current_entity());
   }
   ```
   Note: `NCursesWindowLC_id` is a static variable created by `CEL_Lifecycle(NCursesWindowLC)` in the same TU. `cels_get_current_entity()` returns the entity being composed.

   IMPORTANT: The `cels_lifecycle_bind_entity()` call MUST come AFTER `cel_has(NCurses_WindowConfig, ...)` because the on_create observer reads the config from the entity. If lifecycle binds before the component is added, the observer will find no config.

8. **Remove ALL weak stubs:**
   - Delete `__attribute__((weak)) void ncurses_register_window_observer(void) {}`
   - Delete `__attribute__((weak)) void ncurses_register_window_remove_observer(void) {}`
   - Keep `__attribute__((weak)) void ncurses_register_input_system(void) {}` (still needed -- real impl is in tui_input.c)
   - Keep `__attribute__((weak)) void ncurses_register_frame_systems(void) {}` (still needed -- real impl is in tui_frame.c)
